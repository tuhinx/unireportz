## 🧭 How to Use

1. 🌐 Visit [https://unireportz.com](https://unireportz.com)
2. 🧾 Choose a tool:
   - **Cover Page**
   - **Class Routine**
   - **Exam Schedule**
3. ✏ Fill in:
   - University name
   - Department, course, year
   - Instructor or author name
4. ⬇ Click **Download PDF**
5. ✅ You’re done!
